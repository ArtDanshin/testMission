# Информация

Задача на скачивание, сортировку и вывод данных.
Полностью не была выполнена.
Реализовано получение данных с сервера и вывод их на странице

# Технологии

- HTML (Самый простой)
- CSS (Минимум)
- JavaScript, jQuery
- Bower, npm
- gulp, BrowserSync

# Инструкция

1. Зайти в папку с проектом
2. Выполнить bower i
3. Выполнить npm i
4. Выполнить gulp
5. В браузере откроется страницв
