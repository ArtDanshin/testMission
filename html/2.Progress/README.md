# Информация

Прогресс-бар реализованный с помощью трех блоков. Размер легко изменяется простым изменением размера блока .progress-bar. Заполнение зависит от процентного значения размера дочернего блока .progress-bar__success. Текст зависит от начального значения прогресс-бара и выводится с помощью jQuery.

# Технологии

- HTML
- CSS3
- БЭМ-наэйминг
- JavaScript, jQuery
- Bower

# Инструкция

1. Зайти в папку с проектом
2. Выполнить bower i
3. Запустить index.html
